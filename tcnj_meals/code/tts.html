Computer Accessibility & Context Awareness: Visibility

All webpages must take into account users with disabilities. For those with visibility impairments or similar, and tend to find the text of the webiste to be unclear. Please perform the following steps within the browser to access Text-to-Speech (TTS).

Steps:

  1. Highlight desired line(s) to be translated into TTS
  2. Copy line (CTRL C)
  3. Click TTS button. A prompt will then generate
  4. Paste (CRTL V) line(s) into prompt text-box and press 'OK' when finished
  5. A TTS conversion should be generated

Please Note: After selecting 'CANCEL' when prompt appears, it can be reaccessed by simply pressing the 'Text-to-Speech' button
